pvp-teloot

PVP add-on for Minecraft Fabric with many settings and much more

Version: 1.0.0 Status: Available License: Apache-2.0

Description

pvp-teloot is a comprehensive PVP add-on for Minecraft Fabric. It provides players with a wide range of customizable settings to enhance their PVP experience. Whether you are a casual player or a competitive PVP enthusiast, this mod offers features designed to improve your gameplay.

Features

  • Fully customizable PVP settings
  • Performance optimizations
  • Various gameplay enhancements
  • Compatible with Fabric mods
  • Lightweight and efficient

Installation

  1. Ensure you have Minecraft Fabric installed
  2. Download the latest release from GitHub
  3. Place the JAR file in your mods folder
  4. Launch Minecraft with Fabric

Requirements

  • Minecraft Fabric
  • Fabric API
